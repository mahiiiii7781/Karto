# 🛒 Karto - Hyperlocal Commerce & Delivery Platform

A modern full-stack hyperlocal commerce platform built with React Native and Node.js, designed to connect customers, vendors, riders, and administrators in a single ecosystem.

Karto enables users to order food, groceries, medicines, electronics, gifts, and local store products while providing fast delivery through a dedicated rider network.

## 🚀 Getting Started

### Prerequisites

* Node.js 18+
* React Native CLI
* Android Studio
* PostgreSQL
* Git

### Installation

```bash
git clone <repository-url>
cd karto
npm install
```

### Backend

```bash
cd karto-backend

npm install

npx prisma generate

npx prisma migrate deploy

npm run dev
```

### Mobile App

```bash
cd karto

npm install

npx react-native start

npx react-native run-android
```

## 🌟 About Karto

Karto is not just a food delivery application.

It is a complete hyperlocal commerce ecosystem designed for Tier 2, Tier 3, Tier 4 and Tier 5 cities where customers can order food, groceries, medicines and local products from nearby vendors with fast delivery through a dedicated rider network.

Our mission is to empower local businesses, create opportunities for riders, and provide customers with a fast, reliable, and affordable commerce platform.

Built with ❤️ for the next generation of local commerce.
